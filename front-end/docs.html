About our documentAtIoN

In our documentation, you will find descriptions of each endpoint that exists in the Nicki Min-API, as well as an example AJAX call for that endpoint. Following that, there is an example of the endpoint's output. Enjoy!

NOTE: In queries, replace spaces with dashes and lowercase all words. For example: the album "Pink Friday" would be "pink-friday."

SEARCH FOR A SPECIFIC SONG

/api/songs/:title

With this endpoint, you can query for a specific song by Nicki Minaj. The input is case insensitive, and will return a JSON formatted object with the following properties: title, lyrics, album.

Produce original bars

/api/spit/:bars

Welcome to the reason this project exists. At this endpoint you can supply a number as it's "bars" argument. The returned result will be a string that is "original" bars generated based on the corpus of all of Nicki Minaj's music within the database. It generates the content via Markov Chain, and proves to be a lot of fun. Check out our serenade section to see entire "original" verses.
